鹏鼎控股首次荣登2025年标普全球永续年鉴，彰显可持续发展实力 ​

2月11日，标普全球（S&P Global）2025年《可持续发展年鉴》，鹏鼎控股凭借在环境、社会及公司治理(ESG)方面的优秀表现首次入选标普全球2025年《可持续发展年鉴》。

近年来，凭借在ESG方面持续的资源投入与表现提升，鹏鼎控股在全球ESG评级方面屡获佳绩。鹏鼎控股积极履行社会责任，持续对外发布CSR报告，彰显其对诚信、责任等核心价值观的坚持，以及推动可持续发展和社会和谐的承诺。最新数据显示，鹏鼎控股获晨星Sustainalytics ESG低风险评级；华证AA评级；中证A评级；万得A评级；商道融绿A评级。

作为全球最大的PCB生产企业，鹏鼎控股凭借在产品高端化、布局全球化、生产制造智能化等方面的卓越表现，成功入选标普全球（S&P Global）2025年《可持续发展年鉴》，体现了公司在环境保护、社会责任和治理结构等方面的优异表现。未来，鹏鼎控股将继续致力于可持续发展，为构建绿色、和谐的社会贡献力量。
